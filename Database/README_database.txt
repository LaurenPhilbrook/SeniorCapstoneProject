1. Within your Google App Engine Project, upload the database_definition.sql file to "Cloud Storage".
	a. Click Storage
	b. Click Cloud Storage
	c. Click Storage browser
	d. Click Create bucket
	e. Click Upload files
	f. Upload database_definition.sql
2. Add to database.
	a. Click Storage
	b. Click Cloud SQL
	c. Create New instance
	d. Click on the instance you just created
	e. Click Import
	f. Type in cloud storage path of bucket created above
